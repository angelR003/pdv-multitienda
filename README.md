# PDV MultiTienda Familiar

Sistema para controlar ventas, inventario, traspasos, caducidades y cortes de caja en tres tiendas familiares.

## Objetivo inicial

Crear una primera versión funcional que permita:

- Registrar productos.
- Controlar inventario por tienda.
- Registrar ventas.
- Registrar entradas de mercancía.
- Registrar traspasos entre tiendas.
- Detectar faltantes, sobrantes y productos próximos a caducar.
- Generar cortes básicos de caja.

## Enfoque técnico

- JavaScript
- Node.js
- SQLite
- Interfaz web local
- Funcionamiento offline primero
- Sincronización futura entre tiendas
